# Lab Planner

## Objectif
Organiser les analyses médicales en laboratoire.  
- Priorité : STAT > URGENT > ROUTINE
- Assigner des techniciens et équipements compatibles  
- Éviter les conflits de planning

---

## Installation

Installer les dépendances avec Poetry :

```bash
poetry install
```
## Exécution

```bash
python -m lab_planner.main
```

## Tests

Avec pytest :

```bash
poetry run pytest
```
